# 🚀 Dayanan - Animated Portfolio

A **unique and highly animated portfolio** built with React, Framer Motion, and modern web technologies. This portfolio stands out with its interactive animations, 3D effects, and engaging user experience.

## ✨ Unique Features

### 🎨 **Interactive Particle Background**

- **Canvas-based particle system** with 100+ animated particles
- **Mouse-responsive interactions** - particles react to cursor movement
- **Dynamic connections** between nearby particles
- **Smooth animations** with physics-based movement

### 🔮 **Morphing Geometric Shapes**

- **Continuous shape transformations** between circle, square, and triangle
- **Floating motion** with subtle drift effects
- **Color-shifting animations** based on morphing progress
- **Layered depth effects** for visual richness

### 🖱️ **Custom Interactive Cursor**

- **Dual-cursor system** with main cursor and trailing dot
- **Context-aware interactions** - shows different text for different elements
- **Smooth following animation** with easing curves
- **Hover state detection** for enhanced user feedback

### 🎭 **3D Project Cards**

- **Perspective transforms** that respond to mouse movement
- **Depth layers** with multiple Z-axis positioning
- **Dynamic lighting effects** with glow animations
- **Smooth 3D rotations** based on cursor position

### 🌟 **Creative Loading Animation**

- **Multi-stage loading sequence** with progress indicators
- **Morphing logo animation** with rotating elements
- **Floating particle effects** during loading
- **Smooth transitions** to main content

### 🎪 **Enhanced UI Animations**

- **Scroll-triggered animations** with viewport detection
- **Staggered entrance effects** for content sections
- **Interactive hover states** with 3D transforms
- **Smooth page transitions** and micro-interactions

## 🛠️ **Technologies Used**

- **React 19** - Latest React with modern features
- **Framer Motion** - Advanced animation library
- **Canvas API** - Custom particle and shape rendering
- **CSS3 Transforms** - 3D effects and perspective
- **Vite** - Fast build tool and dev server
- **Lucide React** - Beautiful icon library

## 🚀 **Getting Started**

1. **Clone the repository**

   ```bash
   git clone <your-repo-url>
   cd dayanan
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start development server**

   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
